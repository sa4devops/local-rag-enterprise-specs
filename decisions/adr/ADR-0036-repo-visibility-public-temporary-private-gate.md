# ADR-0036 — ظهور المستودعات: عامة مؤقتاً، والتحويل إلى Private بوابة ما-قبل-الإنتاج

**السياق (Context):** المستودعات الثلاثة (`specs` · ‏`platform` · ‏`aql`) عامة على GitHub أثناء مرحلة الإنهاء والتجربة، والمنتج تجاري مملوك لـAQLORA يتجه لبيئات حكومية ومؤسسية معزولة. قرر المالك (Q4 + ‏DR-1) تقنين هذا الوضع المؤقت وحدوده وبوابة إنهائه.

**الوضع السابق (Previous state):** لا قرار مدوّن لسياسة الظهور؛ ‏`aql/.env` متتبَّع بقيم وهمية؛ ‏`platform/docs/SPEC_SOURCE.md` يصف specs خطأً بأنه «private» (يُصحَّح في دفعة platform الملاصقة)؛ لا ملفات LICENSE في المستودعات؛ ‏D9 يمنع الأسرار في أي طبقة واجهة.

**القرار (Decision):**
1. تبقى المستودعات الثلاثة **عامة مؤقتاً** أثناء الإنهاء والتجربة.
2. **الملكية الفكرية (DR-1 — قرار مالك):** يُعتمد إشعار ملكية مخصص نصه: **«Proprietary — All Rights Reserved © 2026 AQLORA»**؛ **لا تُستخدم أي رخصة Open Source** لأي من المستودعات الثلاثة. ويُوثَّق صراحة: **كون المستودعات عامة مؤقتاً لا يجعلها مفتوحة المصدر ولا يمنح أي ترخيص ضمني، ولا يجعل محتواها سرياً** — فالعلنية المؤقتة قرار تشغيلي لا قرار ترخيص. الصياغة القانونية التفصيلية للإشعار **ليست مانعاً لـG1**، وتُسجَّل حاجتها إلى **مراجعة قانونية قبل الإطلاق التجاري** (بند مؤجلات).
3. **فوراً (بوابة G2):** مسح أسرار شامل لكل التاريخ والفروع في المستودعات الثلاثة؛ إخراج `aql/.env` من التتبع مع إبقاء `.env.example` وهمياً؛ **حظر قاطع** لأي بيانات أو أسرار حقيقية في أي مستودع ما دام عاماً — ويستمر الحظر بعده حتى بوابة الإنتاج.
4. **قبل التحويل:** اختبار قدرة Rocket وCI على مستودع خاص تجريبي، واعتماد نمط تفويض «Only select repositories».
5. **التحويل إلى Private بوابة إلزامية قبل**: أي بيانات حقيقية أو أسرار أو تشغيل إنتاجي — وهو بوابة (G6) لا شرطاً لتعطيل العمل الجاري؛ الترتيب الانتقائي مسموح: ‏specs/platform قبل `aql` رهن نتيجة اختبار Rocket.
6. أثر التحويل (حماية الفروع بخطة GitHub الحالية، صلاحيات CI، الروابط) يُختبر ويوثَّق ضمن البوابة.

**البدائل (Alternatives):** الخصخصة الفورية (مرفوضها المالك — تعطل تجربة Rocket/CI الجارية)؛ رخصة مفتوحة المصدر (مرفوضة — منتج تجاري مملوك)؛ ‏source-available مقيدة (مؤجلة — لا حاجة لها الآن والإشعار الصريح أوضح)؛ البقاء عامة بلا إشعار (مرفوض — يترك النية غامضة رغم أن الافتراض القانوني «كل الحقوق محفوظة»).

**السبب (Rationale):** إشعار صريح يوضح النية للعلن ويحمي العلامة؛ والبوابة الإلزامية تفصل نظيفاً بين «تجربة علنية بلا أسرار» و«تشغيل حقيقي خاص».

**العواقب (Consequences):** ملفات الإشعار (`LICENSE`/ترويسات) تُودَع في G2 لمستودع `aql` (البند 26 من خطة الأصول) وبنفس النص للمستودعين الآخرين في أول دفعة تالية لكلٍّ منهما؛ صف «مراجعة قانونية قبل الإطلاق التجاري» يُضاف لسجل المؤجلات (نفس الدفعة)؛ ‏G6 تقيَّد بهذا القرار وتسبق P8 إلزاماً.

**خطة الانتقال (Migration plan):** ‏G2: المسح + ‏.env + إشعار `aql`؛ نافذة ما بعد G3: اختبار المستودع الخاص التجريبي؛ ‏G6: التحويل بالترتيب الانتقائي + قائمة ما-بعد-التحويل؛ قبل الإطلاق التجاري: المراجعة القانونية للإشعار.
